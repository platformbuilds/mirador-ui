```
Frontend Technologies:
├── Core Framework: React 18 + TypeScript 5.x
├── Build Tool: Vite (faster than Webpack)
├── Visualization: D3.js v7 + Canvas API for performance
├── UI Components: Custom components + Headless UI
├── Styling: Tailwind CSS + CSS Modules
├── State Management: Zustand + TanStack Query (React Query)
├── Routing: React Router v6
├── Testing: Vitest + React Testing Library + Playwright
└── Development: Storybook for component development

Backend Technologies:
├── Runtime: Node.js 20 LTS + Express.js
├── Language: TypeScript
├── API Client: Axios with retry logic
├── WebSocket: Socket.io for real-time data
├── Authentication: JWT + Passport.js
├── Validation: Zod for request/response validation
├── ORM: Prisma for MySQL database access
├── Caching: ioredis for Valkey integration
├── Testing: Jest + Supertest
└── Documentation: OpenAPI/Swagger

Database & Storage:
├── Primary Database: MySQL 8.0 (Dashboard metadata)
├── Cache Layer: Valkey cluster (Session, real-time data)
├── File Storage: Kubernetes persistent volumes
└── Time-Series Data: MIRADOR-CORE (VictoriaMetrics)

Infrastructure:
├── Container: Docker + multi-stage builds
├── Orchestration: Kubernetes 1.27+
├── Load Balancer: HAProxy (Layer 7)
├── Package Manager: Helm 3.x
├── CI/CD: GitHub Actions + ArgoCD
└── Monitoring: Prometheus + Grafana + Jaeger
```
